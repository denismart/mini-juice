# Библиотека mini-juice для работы с VK MiniApps

####Установка с помощью npm:

    npm i mini-juice

####Установка с помощью yarn:

    yarn add mini-juice

####Пример импорта

     import { secondsToTime } from 'mini-juice/common';
     import secondsToTime from 'mini-juice/common/secondsToTime';
